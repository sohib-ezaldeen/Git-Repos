# Git Repos Finder 🔎

A simple web application that allows users to fetch and display public repositories of any GitHub user using the GitHub API.

## 🚀 Live Demo
👉 https://sohib-ezaldeen.github.io/Git-Repos/

---

## 📌 Project Description

This project uses the GitHub API to fetch repositories dynamically based on the username entered by the user.

The application demonstrates:
- Working with APIs
- Using fetch()
- Handling Promises (.then)
- DOM manipulation
- Basic error handling

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- GitHub API
